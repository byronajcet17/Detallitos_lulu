# 🛍️ CampusShop – Frontend E-Commerce Académico

## 📌 Descripción del Proyecto

**CampusShop** es una interfaz frontal de un E-Commerce académico diseñado para la compra y venta de productos dentro del entorno universitario. Este proyecto consiste en la maquetación completa de todas las vistas principales utilizando únicamente **HTML5 y CSS3**, replicando fielmente los mockups y wireframes proporcionados.

El desarrollo sigue un enfoque **mobile-first**, garantizando una experiencia clara, ordenada y responsive. No se utiliza JavaScript, por lo que todas las interacciones son simuladas de forma visual.

---

## 🎯 Objetivos

- Construir una interfaz moderna sin uso de JavaScript  
- Aplicar HTML semántico correctamente  
- Diseñar una experiencia responsive (mobile-first)  
- Replicar con precisión el diseño del mockup

## ⚙️ Especificaciones Técnicas

- HTML5 semántico (`header`, `nav`, `main`, `section`, `article`, `footer`)
- CSS3 puro (sin frameworks)
- Diseño mobile-first
- Media queries para responsividad
- Código modular en múltiples archivos CSS
- Accesibilidad básica (contraste AA)
- Consistencia visual (espaciado, tipografía)

---

## 🎨 Guía de Estilo

- Paleta clara con acentos violeta/azul  
- Cards con sombras suaves  
- Iconografía minimalista  
- Formularios limpios y centrados  

---

## 🚫 Restricciones

- No usar JavaScript  
- No usar frameworks (Bootstrap, Tailwind, etc.)  
- No usar preprocesadores (SASS, LESS)  

---

## 📱 Responsive Design

El diseño sigue el enfoque:

**Mobile → Tablet → Desktop**

Se implementan `media queries` para adaptar la interfaz a diferentes dispositivos.
